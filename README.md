# Nuxt Getting Started

This is a template for a Nuxt project with Genezio. It is simple and easy to use, and it is a great starting point for your nuxt project.

In the `app.vue` file, you will see a very basic Nuxt project.

You can remove welcome page by replacing <NuxtWelcome /> in app.vue with your own code, or creating your own app.vue if it doesn't exist.

# Deploy
:rocket: You can deploy your own version of the template to Genezio with one click:

[![Deploy to Genezio](https://raw.githubusercontent.com/Genez-io/graphics/main/svg/deploy-button.svg)](https://app.genez.io/start/deploy?repository=https://github.com/Genez-io/nuxt-getting-started)


## Genezio CLI Commands

Genezio also provides a CLI tool that you can use to deploy your project from your machine.
All commands are run from the root of the project, from a terminal:

| Command                   | Action                                           |
| :------------------------ | :----------------------------------------------- |
| `npm install -g genezio`  | Installs genezio globally                        |
| `genezio login`           | Logs in to genezio                               |
| `genezio local`           | Starts a local server                            |
| `genezio deploy`          | Deploys a production project                     |
| `genezio --help`          | Get help using genezio                           |
